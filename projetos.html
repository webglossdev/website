Diagnóstico Técnico (Ajuda)

🚧 Em Desenvolvimento
HTML CSS JavaScript JSON Vercel GitHub

O Quê e Como:

Minha página de técnico de TI, junto com uma ferramenta para o cliente achar a estimativa de preço e tempo de serviço antes de me contratar. Feito com HTML, CSS e JS e com deploy no Vercel com GitHub.

Por Que:

A maior parte dos técnicos tem apenas uma página que mostra um link pro whatsapp, assim algumas pessoas se desinteressam e nem sequer entram em contato. Eu queria algo que já desse ao cliente o que ele precisa saber, o que também economiza meu tempo caso alguém queira um tipo de serviço que eu não ofereça.

Milestones:

  • Criar o layout standalone "Aero Glass" e responsivo.
  • Refatorar dados fixos para buscar dinamicamente com manipulação de DOM pelo JavaScript (Fetch API + JSON).
  • Implementar e polir as páginas complementares de "Avaliações" e "Trabalhos".
  • Separar totalmente em um subdomínio (Ex: ajuda.meusite.com).
  • Lançamento final para clientes.

Calendário Olímpico

🌐 Público
Carregando dados do GitHub...
HTML CSS JavaScript JSON (Base de dados) Vercel Github

O Quê e Como:

Um website que centraliza todas as olimpíadas científicas brasileiras do ano: datas de provas e inscrição, matérias que caem, séries que podem participar e links para estudo específico de cada prova.

Por Que:

Eu participava de várias olimpíadas e notei que era muito difícil encontrar todas as informações organizadas em um só lugar. Decidi criar essa ferramenta para ajudar outros estudantes.
